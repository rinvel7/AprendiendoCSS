MODELOS DE CAJA

Modelo de Caja

Es la forma en que CSS ve los elementos de HTML, es decir, en forma de cajas con las siguientes propiedades:

  1. Content: el contenido del elemento HTMl, tomando en cuenta sus dimensiones (width & height)
  2. Margin: distancias externas
  3. border: lo que delimita cada uno de los elementos HTML
  4. padding: (relleno) distancias internas (distancia del borde al contenido

Dentro del modelo de caja hay que considerar 4 lados:

  • Arriba (top)
  • Derecha (right)
  • Abajo (bottom
  • Izquierda (left

Cajas en Linea & Cajas de Bloque


Cajas en Linea


  1. Ocupan el espacio necesario para mostrar su contenido
  2. No tienen dimesiones modificables (alto, ancho)
  3. Permiten otros elementos a su lado
  4. Padding y margin solo empujan a elementos adyacentes en horizontal, UNCA VERTICAL

Cajas de Bloques


  1. Ocupan todo el ancho disponible, lo que genera saltos de linea
  2. No tienen dimesiones modificables (alto, ancho)
  3. No Permiten otros elementos a su lado (aunque especifique un ancho, siguen ocupando todo el espacio disponible a lo ancho, generando saltos de linea

Propiedad Display (Visualización)


Establece los tipos de visualización interna y externa de un elemento. Nos permite definir como será mostrado un cierto elemento HTML.


Propiedades:

  1. inline
  2. block
  3. inline-block (se comporta como un elemtento en linea pero acepta modificar sus dimensiones (alto y ancho)
  4. none
  5. table
  6. list-item
  7. flex
  8. grid

Propiedad Overflow (Desbordamiento)

Define cómo se muestra el contenido desbordado tanto en el eje horizontal como en el vertical.

Propiedades:

  1. Visible (defaul)
  2. hidden (Oculto)
  3. Scroll (barra de desplazamiento vertical
  4. Auto (el navegador decide si mostrar una barra de desplazamiento o no)
