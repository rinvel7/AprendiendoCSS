Algoritmo de CSS

Es la forma enque el navegador aplica los estilos al documento HTML. Es importante entender este concepto para no sobreescribir las reglas del CSS

  • La Cascada
  • La Especificidad
  • La Herencia

La Cascada

La cascada es el mecanismo que tiene el navegador para ir aplicando los estilos, y se dividen en tres aspectos:

El origen del codigo:
User Agent
Son estilos predeterminados del navegador le da a las etiquetas.
Personalizacion
Es aquellas que el usuario le asigna al navegador o sistema operativo. Ej: cuando entra a configuracion del navegador.
Estilos de autor:
Es Codigo Css, que cargamos en las hojas de estilo.

Orden de aparicion

Es la forma como el navegador va leyendo las lineas de codigo. Los estilos que van quedando, son los que al final se ha colocado.

La Espificidad

Es el peso que tiene un selector cuando hay conflicto de estilos. Es ver cual elemento de un mismo valor a una misma propiedad tiene un mayor peso, y se clacula de la siguiente forma:

  1. Etiquetas y pseudoelementos-------0,0,0,1
  2. Clases, atributos y pseudoclases--0,0,1,0
  3. Identificadorres------------------0,1,0,0
  4. Estilos en linea------------------1,0,0,0
  5. important-------------Rompe la especificidad

La Herencia

Es la capacidad de un selector de obtener(heredar) los valores de sus ancestros mas cercano, para aplicarla se usa el valor inherit, su queremos evitarla podemos asiganar otro valor o inicializar la propiedad en cuestion con el valor initial. Valores que se herdan pueden ser:

  3. la familia tipografica
